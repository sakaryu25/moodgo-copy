## ⏸ logic（保留＝DB実査が必要。安全に直すにはservice_roleでタグ確認要）
1. **タグ空/adminの店がrank1-2に浮上**【影響:高】
   - 例: 「PACHA CRAFT BEER TACOS」(tags空・admin・2.81km)が居酒屋/カフェ/洋食の3検索でrank2。
     「CAFÉ デイジー」(tags空)がまったりカフェ横浜rank1、「三井アウトレット」(tags空)がモール横浜rank1。
   - 原因候補: (a)名前が「BEER」等で居酒屋recall語に緩く一致、(b)admin店が広タグで通る。
   - 直し方: PACHA等の実タグ/通過ゲートをDBで特定 → recall語を狭める or タグ空adminをsortで降格。
     ※親ジャンル選択時は名前一致の正規店(タグ無)も居るため「タグ空一律降格」は不可。ゲート特定が先。
2. **集中/カフェ作業で遠い店が近いスタバを飛び越す（距離逆転）**【影響:高】
   - cafe-shuchu-tokyo: rank1=6.4km(渋谷)、700mのエクセルシオールはrank7。
   - 原因: サブタグ加点が距離減衰を上回る。distance weightの再チューニング要（回帰リスク高→要慎重）。

- **#ショッピング** → リフォーム店/修理店

## 🏷 data（タグ掃除＝要 .env.retag service_role鍵ローテ後。apply_tag_fixes.py / fix_relax_cafes.py で実行）
過剰タグの店を【店名の正規表現】→【除去タグ】で精密化済み。鍵ローテ後そのまま流せる:
- **#体動かしたい/#ガッツリ運動 剥離**: `/(整体|整骨|接骨|鍼灸|マッサージ|リラク|エステ|ネイル|ホットヨガ|ヨガスタジオ|ヨガ教室|ピラティス|岩盤浴|サウナ)/` かつ運動語を含む店（実測: ヨガ67/ピラ55/整体8等）
- **#ショッピング系 剥離**: `/(リフォーム|お直し|裾上げ|靴修理|時計.*修理|宝石.*修理|合鍵|クリーニング|コインランドリー|買取専門|質屋)/` → 除去[#ショッピング,#服アクセサリー,#雑貨インテリア,#古着,#コスメ美容,#お土産]
- **#スリル味わいたい 剥離**(source=ghostmap優先): `/(公衆トイレ|便所|トイレ$|公衆電話|電話ボックス|自販機|バス停|議事堂|^皇居|駅$|駐車場$)/` → 除去[#スリル味わいたい,#心霊スポット,#パワースポット]（心霊トンネル/廃墟/駅跡/墓は語末条件で温存）
- **#パワースポット 剥離**: `/(駐車場$|パーキングエリア|公衆トイレ|便所|バス停|入口$|参拝者駐車|有料駐車)/` → 除去[#パワースポット,#まったりしたい]
- **#ドライブしたい 剥離**: `/(運河$|大学.*グラウンド|女子大.*グラウンド|用水路|排水路)/` → 除去[#ドライブしたい,#絶景スポット,#自然感じたい]
- **#わいわい楽しみたい 剥離**: `/画廊/` かつ (美術館|博物館|記念館)非含 → 除去[#わいわい楽しみたい]（#鑑賞は残す）
- **#温泉 剥離(ゴルフ場)**: fix_relax_cafes.py の`カントリークラブ`限定を`/ゴルフ|カントリー倶楽部|CC$|ゴルフ場|パークゴルフ|打ちっぱなし/`かつ温浴語非含に一般化
- **#犬カフェ 剥離**: fix_relax_cafes.py の2店固定を`/&\s*bar|DEPT|ウェア|アパレル|用品|グッズ|トリミング|ペットサロン|動物病院|ホットドッグ/`一致で剥離に一般化
- **#景色良いカフェ 剥離**: #海辺カフェ blanket剥離ロジックを拡張。景観語(展望|スカイ|ルーフ|テラス|海|湖|山|夜景|view|[0-9]+階)を名前・住所に一切含まないadmin/osm店から剥離
- **#森の中で深呼吸 格下げ**: 大型自然タグ(#大型公園|#自然公園|#森|#渓谷)非併設の小緑地(街区公園/児童遊園/住宅地内◯◯緑地)を森系対象から除外
- **#韓国 付与**(掃除でなく補完): 韓国チキンチェーン(bb.q/ネネチキン/チョンガーネ/クリスピーチキン)に#韓国付与→reliableTrust直通
